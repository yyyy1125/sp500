# sp500
S&P500 分钟线时序预测全栈系统
基于 RNN / GRU / LSTM 的「云端训练 - 边缘推理 - 前端可视化」深度学习教学实验平台
本项目针对混沌金融市场低信噪比、随机游走的特性，实现了一套完整的标普 500 分钟线价格预测全栈系统。通过可视化波形对比，直观呈现循环神经网络的时序记忆能力、梯度消失现象与过拟合规律，打通从模型训练、边缘部署到前端交互的完整工程链路。
✨ 核心特性
三模型统一框架：原生支持 RNN / GRU / LSTM 一键切换，可横向对比参数量、预测精度与相位滞后差异
鲁棒性训练机制：集成 Dropout、权重衰减、时序划分验证集、早停机制，抑制模型拟合市场噪声
断点续训能力：适配共享 GPU 环境，支持信号捕获与训练进度自动保存恢复
流式边缘推理：WebSocket 逐点推送行情与预测结果，模拟工业级分钟级遥测延迟
可视化示波器：基于 D3.js 打造暗黑工业风监控面板，实时呈现真实行情、预测波形、绝对误差与系统延迟
无额外依赖推理：推理端脱离 scikit-learn，手动实现归一化 / 反归一化，适配低算力边缘设备

🛠️ 技术栈
深度学习框架：PyTorch
数据处理：NumPy、scikit-learn（仅训练端使用）
通信协议：WebSocket
前端可视化：D3.js v7
运行环境：Python 3.8+
🚀 快速开始
1. 安装依赖
pip install torch numpy scikit-learn websockets
2. 执行模型训练
python sp500_train.py
修改脚本顶部 MODEL_TYPE 可切换 RNN / GRU / LSTM
训练完成后自动生成对应权重文件与归一化参数文件
3. 启动本地推理服务
python pythonserver.py
服务默认监听 ws://127.0.0.1:8765
修改脚本顶部 MODEL_TYPE 可加载对应模型的权重文件
4. 打开前端可视化
直接双击 index.html 在浏览器中打开，即可查看实时行情预测示波器。
🧪 实验教学点
本项目可直观复现以下深度学习与时序建模核心知识点：
梯度消失可视化：对比 RNN 与 LSTM 的预测波形，观察 RNN 的相位滞后与振幅衰减现象
隐状态作用验证：推理时清零隐状态，观察预测波形失去时序记忆的表现
参数量性价比：横向对比 RNN / GRU / LSTM 的总参数量与预测误差差异
过拟合与正则化：调整 Dropout、权重衰减、早停参数，观察验证集损失变化
工程部署权衡：修改 hidden_size，观察模型精度与推理延迟的博弈关系
